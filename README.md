# 🛒 MarketBaseX - Full-Stack MERN E-Commerce Platform

MarketBaseX is a robust, production-ready e-commerce solution designed for high performance and seamless user experience. Built with the **MERN** stack, it features a scalable backend, a dynamic frontend, and full administrative control.

## ✨ Core Features

### 👤 User Experience

- **Dynamic Storefront:** Browse products with real-time filtering and category navigation.
- **Secure Authentication:** JWT-based login and signup with HTTP-only cookies for enhanced security.
- **Shopping Cart:** Persistent cart functionality with real-time price calculations.
- **Checkout & Payments:** Secure payment processing integrated with **PayPal Sandbox**.

### 🛠️ Administrative Control

- **Product Management:** Full CRUD (Create, Read, Update, Delete) capabilities for products and categories.
- **Order Tracking:** Monitor and update order statuses in real-time.
- **Media Handling:** Seamless image uploads and hosting via **Cloudinary API**.
- **User Management:** Oversee user accounts and assign administrative roles.
